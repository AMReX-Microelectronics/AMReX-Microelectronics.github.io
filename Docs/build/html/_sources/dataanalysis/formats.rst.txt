Output Formats and Visualization
==============

All MicroEleX code packages write the output data in `plotfile format <https://amrex-codes.github.io/amrex/docs_html/IO.html>`__

Plotfiles are AMReX' native data format.

Refer to `AMReX documentaion <https://amrex-codes.github.io/amrex/docs_html/Visualization_Chapter.html>`__ for several visualization tools that can be used for AMReX plotfiles.
